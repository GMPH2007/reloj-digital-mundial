🚀 **Chronos** es un reloj digital mundial premium de última generación que combina la simulación horaria global interactiva con herramientas de precisión y una bóveda de seguridad militar de conocimiento cero. Diseñado con una estética cyberpunk de alto impacto visual y optimizado para una alta visibilidad (SEO) en internet.

## 🌟 Características Destacadas

### 1. 🗺️ Mapa de Fronteras Interactivo
* Visualiza la división geopolítica del planeta en un mapa vectorial HD.
* Pasa el mouse para iluminar las fronteras de los países en neón.
* Haz clic en cualquier país del mundo para añadir automáticamente la hora de su capital al panel.
* Los países seleccionados se quedan encendidos en el mapa con un borde de neón fijo.

### 2. 🛡️ Bóveda de Seguridad Cero Conocimiento (Zero-Knowledge Vault)
* Protege tus datos de alarmas y relojes preferidos.
* Cifra toda tu configuración local usando **AES-GCM de 256 bits** y derivación de clave **PBKDF2** con 100,000 iteraciones (mediante la Web Crypto API).
* Al activarse, la base de datos se guarda cifrada. Si el navegador es reiniciado, una pantalla de bloqueo con teclado numérico táctico y consola terminal bloqueará el acceso hasta que ingreses tu PIN.
* **Seguridad Absoluta**: La contraseña nunca se almacena, por lo que ni el creador del código ni un hacker pueden desencriptar los datos sin la contraseña.

### 3. 📊 Planificador de Reuniones Globales
* Convierte husos horarios fácilmente.
* Desliza el control horaria y calcula de inmediato el estado del día (Laboral, Personal o Sueño) para todas tus ubicaciones agregadas, simplificando llamadas internacionales.

### 4. ⏱️ Herramientas de Precisión
* **Alarmas Programadas**: Define alertas con tonos programáticos de audio generados mediante la API de síntesis Web Audio (sin archivos externos).
* **Cronómetro**: Precisión de milisegundos y registro de vueltas ("laps").
* **Temporizador**: Cuenta regresiva con anillo animado de progreso de tiempo.

### 🏷️ Pegatinas de Información ("Stickers")
* ¿No sabes para qué sirve cada tarjeta? Hemos colocado un botón interactivo `?` en cada panel. Al hacerle clic, despliega una "pegatina" holográfica con la descripción detallada del módulo.

## 🛠️ Tecnologías Utilizadas
* **Estructura**: HTML5 semántico con tags optimizados para SEO y accesibilidad.
* **Diseño**: CSS3 Vanilla (Glassmorphic variables, Neon borders, Keyframes animaciones).
* **Simulación**: JavaScript nativo (Intl API para relojes sin errores de parseo, Web Audio API para osciladores de sonido, Web Crypto API para cifrado AES-256).
